# Gravity Simulator



My first Python physics simulation project.

## Features

- Gravity simulation
- Orbital motion
- Trail visualization
- Speed indicator
- Distance indicator
- Pause / Resume
- Reset
- Basic OOP structure

## Technologies

- Python
- Pygame

## Controls

- Space: Pause / Resume
- R: Reset simulation
